# デザイントークン・画面ビジュアル仕様

ワイヤーフレーム [`docs/mockups/家計簿ワイヤーフレーム.html`](../mockups/家計簿ワイヤーフレーム.html) から抽出した、SwiftUI 実装時に使用する具体的なデザイン仕様。

ライト / ダークの両モードに対応する。

## カラーパレット

### Light Mode

| ロール          | 16 進     | SwiftUI 推奨                                | 用途                               |
| --------------- | --------- | ------------------------------------------- | ---------------------------------- |
| ink             | `#1d1d1f` | `Color.primary`（システムフォント色と整合） | プライマリテキスト・基本アイコン   |
| inkSoft         | `#5a5a5f` | `Color.secondary`                           | セカンダリテキスト                 |
| inkLight        | `#9c9ca1` | `Color(.tertiaryLabel)`                     | 入力プレースホルダ・無効状態       |
| paper           | `#fafaf6` | アプリ背景                                  | 通常画面の背景                     |
| paperWarm       | `#f5f1ea` | カード背景・分離フィル                      | カード・セグメント未選択           |
| accent (orange) | `#ff6b3d` | カスタム Color asset                        | 選択日マーカー、強調アクション     |
| blue            | `#3a82f7` | `Color.blue`（システム）                    | ナビゲーション、リンク、編集ボタン |
| green           | `#34c759` | `Color.green`                               | 収入・予算消化未満                 |
| red             | `#ff3b30` | `Color.red`                                 | 予算超過・破壊的アクション         |

### Dark Mode（iOS 標準ダークパレット準拠）

| ロール    | 16 進                   | 用途                   |
| --------- | ----------------------- | ---------------------- |
| ink       | `#f2f2f7`               | プライマリテキスト     |
| inkSoft   | `#a8a8ad`               | セカンダリ             |
| inkLight  | `#6c6c70`               | 三次・無効             |
| paper     | `#000000`（true black） | アプリ背景             |
| paperWarm | `#1c1c1e`               | グルーピングカード背景 |
| paperDeep | `#2c2c2e`               | 三次フィル             |
| accent    | `#ff7d54`               | 選択日・強調           |
| blue      | `#0a84ff`               | ナビ・リンク           |
| green     | `#30d158`               | 収入                   |
| red       | `#ff453a`               | 予算超過               |
| divider   | `rgba(84,84,88,0.65)`   | 区切り線               |

実装方針: `Assets.xcassets/Colors/` にカラーアセットを定義し、Light/Dark の両値を保持する。SwiftUI からは `Color("ink")` / `Color("paper")` 等で参照。

## タイポグラフィ

| ロール             | サイズ   | ウェイト                 | 用途                                           |
| ------------------ | -------- | ------------------------ | ---------------------------------------------- |
| Page Title (large) | 22pt     | 700, letter-spacing -0.4 | 各画面のタイトル                               |
| Amount (display)   | 38pt     | 700, letter-spacing -1   | 入力画面の大きな金額表示                       |
| Amount (card)      | 18pt     | 700                      | 予算カード・ヘッダーサマリ                     |
| Body               | 12〜14pt | 400 / 500                | 一般テキスト・行内容                           |
| Caption            | 9〜11pt  | 400                      | カテゴリ名・補足                               |
| Label              | 8〜9pt   | 600, letter-spacing 0.3  | 「支出」「収入」「差額」等のセクション小見出し |

フォントは **システムフォント** (`-apple-system` / SwiftUI の `.system(size:)`) を使用。SF Pro が自動選択される。

## タブバー（5 タブ）

画面下部に常駐する `TabView`。

| 順序 | 名前       | SF Symbol（推奨） |
| ---- | ---------- | ----------------- |
| 1    | 入力       | `plus.circle`     |
| 2    | カレンダー | `calendar`        |
| 3    | レポート   | `chart.bar`       |
| 4    | 予算       | `target`          |
| 5    | 設定       | `gearshape`       |

- 選択中タブ: `accent` 色（ライトでは orange）
- 非選択: `inkLight`
- 背景: `rgba(255,255,255,0.96)` + `backdrop-filter: blur(20px)`（半透明・すりガラス）

## 画面別ビジュアル仕様

### S-01 入力画面

- 上部: タイトル「入力」(large) + 右上「保存」ボタン（blue）
- セグメンテッドコントロール: 支出 / 収入（デフォルト: 支出、選択肢は白背景 + 影）
- 大きな金額表示: `¥0`（38pt, 700, letter-spacing -1, 中央揃え）
- フィールドカード（白背景, 角丸 10）:
  - カテゴリ → `食費 ›`
  - 日付 → `今日（4月28日）›`
  - メモ → `—`（プレースホルダーは `inkLight`）
- テンキー（3×4 グリッド, gap 5）:
  - 1, 2, 3
  - 4, 5, 6
  - 7, 8, 9
  - **000**, 0, ⌫（削除）
  - 各ボタン: 白背景, 角丸 8, 17pt, 500, 影 `0 1px 2px rgba(0,0,0,.05)`

### S-02 カレンダー画面

#### ヘッダー部

- 左: 「2026年」（11pt, red, 600）+ 「4月」（22pt, 700）の 2 段組
- 右: `‹` `今日` `›`（14pt, blue, gap 14）

#### 月次サマリ（横 3 並び、warm 背景カード）

- 支出: `¥38,420`（ink）
- 収入: `¥250,000`（green）
- 差額: `+¥211,580`（blue）
- 各ラベル「支出」「収入」「差額」: 8pt, inkSoft, letter-spacing 0.3

#### 曜日ヘッダー

- 日（red）/ 月〜金（inkLight）/ 土（blue）, 9pt

#### 日付グリッド

- 7 列 grid, 各セル高 30
- 日番号: 11pt, 500
  - 日曜: red / 土曜: blue / 平日: ink
- 選択日: 直径 17 の角丸正方形に accent 背景, 白文字
- 各セル下に当日支出: 8pt, inkSoft（収入なら緑 + プラス記号）
- ¥9,999 を超える場合は `k` 表記（例: `250000` → `250k`）

#### 日別詳細（divider の下）

- 「4月28日 火曜」（11pt, ink, 600）+ 「合計 ¥3,360」（11pt, inkSoft）
- 各行: 6×6 ドット + メモ（12pt, ink）+ カテゴリ（9pt, inkSoft）+ 金額（12pt, ink, 500）
- 行間 divider: `rgba(inkLight, 30%)` 0.5px

#### ジェスチャ

- シングルタップ: `selectedDate` 更新 → 下半分が当該日のリストに変化
- ダブルタップ: 当該日を初期値とした `TransactionEntrySheet` を起動

### S-03 レポート画面

- ヘッダー: 「2026年 4月」（22pt, large）+ `‹ ›` ナビゲーション
- 3 カード（横並び）: 支出 / 収入 / 差額
- ドーナツチャート (120×120):
  - 内側半径相当: stroke-width 14
  - 中心テキスト「支出 ¥38k」
  - スライス色: 食費 = accent / 交通費 = blue / 娯楽 = `#a06bff` / 日用品 = green / その他 = inkSoft
- 内訳リスト（白背景カード）: 9×9 色キューブ + カテゴリ名 + 割合（%）+ 金額

### S-04 予算画面

- 右上: 「編集」ボタン（blue）
- 全体予算カード:
  - ラベル「全体予算（4月）」（9pt, inkSoft, 600）
  - 金額: `¥38,420`（18pt, 700）/ 限度: `/ ¥80,000`（11pt, inkSoft）
  - プログレスバー: 高さ 6, 角丸 3, 背景 `paperWarm`, 内側 `green`（超過時 `red`）
  - 下部: 「消化 48%」「残り ¥41,580」（9pt, inkSoft, 両端揃え）
- カテゴリ別予算リスト:
  - 各行: カテゴリ名 (11pt, 500) + `¥使用額 / ¥予算`（10pt, 超過時 red）
  - プログレスバー: 高さ 4, 角丸 2
  - 超過時: `⚠ 超過 ¥xx,xxx`（9pt, red）
- 末尾に「+ カテゴリ別予算を追加」（blue）

### S-05 設定画面

メニューは 2 項目のみ（カテゴリ管理 / 固定費の設定）。

- 各行（白背景カード, 角丸 10）:
  - 左: 26×26 角丸アイコン
    - カテゴリ管理: `accent` 背景 + tag アイコン (SF Symbol: `tag`)
    - 固定費の設定: `blue` 背景 + repeat アイコン (SF Symbol: `arrow.triangle.2.circlepath`)
  - 中央: ラベル（14pt, ink）
  - 右: 件数（13pt, inkSoft）+ `›`（14pt, inkLight）

### S-06 収支入力シート

S-01 入力画面と同じレイアウトを `sheet` で表示。`init(draft:)` で初期値（日付、編集なら既存値）を渡す。

### S-07 カテゴリ管理画面

設定からプッシュ遷移。一覧は iOS 標準 `List`（`.insetGrouped` style）。並び替え対応。使用中（参照あり）のカテゴリは削除ボタンを無効化。

### S-08 固定費の設定画面

設定からプッシュ遷移。一覧は `List`。各行: 名称 + 金額 + カテゴリ + 想定支払い日。Transaction の自動生成は行わない（要件定義書「完全手動入力」方針）。

## 共通パターン

### カード

- 背景: 白（Light）/ `paperWarm`（Dark）
- 角丸: 10
- マージン: 左右 12pt
- 内側パディング: 7〜14pt
- 行間 divider: 0.5px, `inkLight` 30% 透過

### ボタン

- ナビゲーション系（保存・編集・追加）: blue, 14pt, 500, 角丸なしのテキストボタン
- ピル型 CTA は当面使用しない（iOS 標準的な見た目を優先）

### アニメーション

- 月切り替え: SwiftUI `.transition(.slide)` または `withAnimation(.easeInOut(duration: 0.25))`
- シート起動: 標準 `.sheet` のデフォルト遷移
- 選択日変更: `.animation(.spring(response: 0.3))` でセル背景色と下半分リストをアニメーション

## 注意事項

- **palt は使わない**（iOS 標準フォントは日本語字詰めを内蔵）
- ヒラギノ角ゴは **Pro**（ProN ではない）
- ダーク背景のテキストに純粋な `#fff` を使わず `#f2f2f7` を使う
- 金額表示は必ず `CurrencyFormatter`（純粋ロジック struct）経由で `¥1,234` 形式に変換する
- SF Symbols は最新版を使用（iOS 17+ で利用可能なもの）
