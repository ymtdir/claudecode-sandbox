# プロダクト要求定義書 (PRD)

## 1. プロダクト概要

### 名称

**UnifiedCal** - タスク管理愛好家のための統合カレンダーアプリ

### プロダクトコンセプト

1. **完全統合型の時間管理**: カレンダー、Todo、リマインダーを一つの時間軸上で管理
2. **タスク管理の楽しさ**: 達成感の可視化とゲーミフィケーション要素
3. **クロスプラットフォーム対応**: iOS/Android両OSで統一された体験を提供

### プロダクトビジョン

タスク管理を愛する人のために、カレンダー、Todo、リマインダーが完全に一体化したクロスプラットフォームアプリを提供する。
Yahoo!カレンダーやTimeTreeの親しみやすいUIを踏襲しながら、タスク管理自体を楽しい体験に変える。
完了したタスクを可視化することで達成感を提供し、統計レポートで自己改善をサポートする。
iOS/Android両方のユーザーに、プラットフォームの違いを感じさせない一貫した体験を実現する。

### 目的

- 複数アプリ間の切り替えを完全に排除し、タスク管理の効率を30%向上させる
- 完了タスクの可視化により、ユーザーの達成感と継続率を向上させる
- タスク管理愛好家が本当に求める機能を、美しいUIで提供する
- iOS/Android両プラットフォームで一貫した体験を提供する

## 2. ターゲットユーザー（ペルソナ）

### プライマリーペルソナ: タスク管理愛好家の田中さん（32歳）

**基本属性**

- 職業: プロジェクトマネージャー（IT企業）
- 経験: タスク管理歴8年、様々なツールを試した経験あり
- デバイス: iPhone 15 Pro、Androidサブ機（Pixel）、iPad Air

**技術スタック・利用ツール**

- 過去に試したツール: Notion、Todoist、Things 3、純正カレンダー+リマインダー
- 現在: 純正アプリの組み合わせに不満を持ちながら使用
- その他: Slack、GitHub、Figma

**現在の課題**

- 純正アプリ間の連携はあるがUXが分散していて使いにくい
- タスクの達成度や統計が見えず、振り返りができない
- 複数アプリを切り替えるたびに集中力が途切れる
- タスク管理自体をもっと楽しくしたい

**期待する解決策**

- すべてを一つのアプリで完結させたい
- Yahoo!カレンダーのような親しみやすいUIがほしい
- 達成度の可視化とゲーミフィケーションで楽しく管理したい
- 詳細な統計レポートで自己改善したい

**1日の典型的なワークフロー**

```
7:00  起床、今日のタスク確認
8:30  通勤中にタスク整理
9:00  朝会でチームのタスク確認
10:00 タイムブロッキングで作業開始
12:00 昼休み、午後のタスク調整
15:00 進捗確認、優先度調整
18:00 今日の達成度確認
21:00 明日のタスク準備
```

### セカンダリーペルソナ: 生産性を追求する鈴木さん（35歳）

**基本属性**

- 職業: フリーランスエンジニア
- 複数プロジェクト並行管理
- 時間管理が収入に直結

**現在の課題**

- タスクと時間の関係が見えにくい
- プロジェクトごとの時間配分が難しい
- 振り返りのためのデータが不足

**期待する解決策**

- タイムブロッキングとタスクの完全統合
- プロジェクト別の時間分析
- AIによるスケジュール最適化

## 4. 機能要件

### 4.1 コア機能（P0: MVP必須）

#### 統合カレンダービュー

**ユーザーストーリー**
タスク管理愛好家として、すべての予定とタスクを一画面で確認するために、統合カレンダービューが欲しい

**受け入れ条件**

- [ ] 月表示でカレンダー、Todo、リマインダーが統合表示される
- [ ] 各日付にタスク数のバッジが表示される
- [ ] 週表示に横スワイプで切り替えられる
- [ ] 週表示では時間軸（0-24時）でスケジュールが表示される
- [ ] ピンチアウトで表示モードを切り替えられる
- [ ] 今日のタスクリストが下部に常時表示される
- [ ] 六曜・祝日が表示される（日本向け）

**優先度**: P0（必須）

#### スマートTodo機能

**ユーザーストーリー**
タスク管理愛好家として、素早く直感的にタスクを管理するために、スマートなTodo機能が欲しい

**受け入れ条件**

- [ ] テキスト入力でタスクを追加できる
- [ ] スワイプジェスチャーで完了/延期ができる
- [ ] 完了タスクが取り消し線で表示され、消えない
- [ ] タスクにカテゴリーを設定できる
- [ ] 期限と優先度を設定できる
- [ ] ドラッグ&ドロップで日付変更ができる

**優先度**: P0（必須）

#### iOS統合リマインダー

**ユーザーストーリー**
iPhoneユーザーとして、適切なタイミングで通知を受けるために、スマートなリマインダー機能が欲しい

**受け入れ条件**

- [ ] 時間ベースの通知を設定できる
- [ ] 位置情報ベースの通知を設定できる（ジオフェンシング）
- [ ] 繰り返し設定ができる（毎日/毎週/カスタム）
- [ ] スヌーズ機能が使える
- [ ] Focus Modeと連携できる
- [ ] 通知の重要度を3段階で設定できる

**優先度**: P0（必須）

#### ビジュアルフィードバック

**ユーザーストーリー**
タスク管理愛好家として、達成感を感じるために、視覚的なフィードバックが欲しい

**受け入れ条件**

- [ ] タスク完了時に振動フィードバックが動作する（iOS/Android）
- [ ] 完了タスクに✓マークとスタンプが表示される
- [ ] 今日の達成率がプログレスバーで表示される
- [ ] 連続達成日数（ストリーク）が表示される
- [ ] タスク完了時にアニメーションが再生される

**優先度**: P0（必須）

### 4.2 重要機能（P1）

#### 共有・コラボレーション機能

**ユーザーストーリー**
複数人でタスクを管理する人として、家族やチームとカレンダーを共有するために、コラボレーション機能が欲しい

**受け入れ条件**

- [ ] カレンダーを他のユーザーと共有できる
- [ ] 共有レベルを設定できる（閲覧のみ/編集可能）
- [ ] 家族やチームメンバーを招待できる
- [ ] タスクの担当者を割り当てられる
- [ ] 共有カレンダーの色分けができる
- [ ] プライベートタスクを非表示にできる

**優先度**: P1（重要）

#### Apple Watchコンパニオンアプリ

**ユーザーストーリー**
Apple Watchユーザーとして、手首から素早くタスクを確認・管理するために、Watch アプリが欲しい

**受け入れ条件**

- [ ] コンプリケーションで今日のタスク数を表示
- [ ] タスクの完了チェックができる
- [ ] 次の予定を確認できる

**優先度**: P1（重要）

#### ウィジェット機能

**ユーザーストーリー**
iPhoneユーザーとして、アプリを開かずに情報を確認するために、ウィジェットが欲しい

**受け入れ条件**

- [ ] ホーム画面ウィジェット（S/M/L）が提供される
- [ ] ロック画面ウィジェットが提供される
- [ ] Smart Stackに対応する
- [ ] ウィジェットから直接タスク追加ができる

**優先度**: P1（重要）

#### 統計ダッシュボード

**ユーザーストーリー**
タスク管理愛好家として、自己改善のために、詳細な統計情報が欲しい

**受け入れ条件**

- [ ] 日/週/月の完了タスク数が表示される
- [ ] カテゴリー別の時間配分が表示される
- [ ] 生産性の高い時間帯が分析される
- [ ] グラフで視覚的に表示される
- [ ] CSVエクスポートができる

**優先度**: P1（重要）
