# 要求内容

## 概要

UnifiedCalのメイン画面となるカレンダー月表示画面を実装します。react-native-calendarsを使用して、タスク数表示、日本の六曜・祝日表示、今日のタスクリストを含む、ユーザーフレンドリーなカレンダー画面を構築します。

## 背景

UnifiedCalのコアとなる機能として、カレンダーとタスクを統合した画面が必要です。ユーザーは一つの画面で月間のタスク概要を確認し、その日のタスクを管理できる必要があります。タスク管理愛好家のユーザーが達成感を感じられるよう、視覚的なフィードバックも重要です。

## 実装対象の機能

### 1. カレンダー月表示機能
- react-native-calendarsを使用したカレンダー表示
- 各日付にタスク数をバッジで表示
- 今日の日付をハイライト表示
- 日本の六曜・祝日表示
- 月の切り替え（左右スワイプまたは矢印ボタン）

### 2. 今日のタスクリスト
- カレンダー下部にタスクリスト表示
- タスクの完了/未完了状態表示
- 完了率をプログレスバーで可視化
- タスクの追加ボタン配置

### 3. スワイプジェスチャー機能
- タスクを右スワイプで完了
- タスクを左スワイプで延期
- カレンダーを左右スワイプで月変更
- ピンチイン/アウトで表示モード切替（月/週）

### 4. ビジュアルフィードバック
- タスク完了時のアニメーション
- 完了タスクの取り消し線表示
- 振動フィードバック（プラットフォーム対応）

## 受け入れ条件

### カレンダー月表示機能
- [x] カレンダーが月表示で正しく表示される
- [x] 日本の六曜が各日付に表示される
- [x] 祝日が赤色で表示される
- [x] 各日付にタスク数バッジが表示される
- [x] 今日の日付が視覚的に識別できる

### 今日のタスクリスト
- [x] 選択した日付のタスクがリスト表示される
- [x] タスクの完了状態が表示される
- [x] タスク完了率がプログレスバーで表示される
- [x] タスク追加ボタンが機能する

### スワイプジェスチャー機能
- [x] タスクの右スワイプで完了マークが付く
- [x] タスクの左スワイプで延期処理が動作する
- [x] カレンダーの左右スワイプで月が切り替わる

### ビジュアルフィードバック
- [x] タスク完了時にアニメーションが再生される
- [x] 完了したタスクに取り消し線が表示される
- [x] プラットフォーム対応の振動フィードバックが動作する

## 成功指標

- カレンダー表示から任意の日のタスク表示まで2タップ以内
- タスク完了操作が1ジェスチャーで完結
- アニメーションが滑らかに動作（60fps維持）

## スコープ外

以下はこのフェーズでは実装しません:

- カレンダーの週表示・日表示への切り替え（基本構造のみ準備）
- タスクの編集機能（別Issue）
- 共有カレンダー機能
- AIによるスケジュール最適化
- 統計ダッシュボード連携
