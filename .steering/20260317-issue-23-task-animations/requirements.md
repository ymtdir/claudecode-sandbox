# 要求内容

## 概要

タスク完了時のユーザー体験を向上させるため、アニメーションとビジュアルフィードバックを実装します。React Native Reanimatedを使用して、滑らかで楽しいアニメーションを作成します。

## 背景

現在のタスク管理アプリケーションでは、タスクの完了時に視覚的なフィードバックが不足しており、ユーザーエクスペリエンスが静的で味気ないものになっています。アニメーションを追加することで、タスク完了の達成感を演出し、ユーザーのモチベーション向上を図ります。

## 実装対象の機能

### 1. チェックマークアニメーション
- SVGパスアニメーションを使用したチェックマークの描画
- タスク完了時に滑らかに表示されるアニメーション

### 2. 波紋エフェクト
- タップ位置から広がる波紋アニメーション
- 色と透明度が徐々に変化する視覚効果

### 3. スタンプアニメーション
- タスク完了時に表示される「完了」スタンプ
- スケールとローテーションを組み合わせた楽しい演出

### 4. 取り消し線アニメーション
- 完了したタスクのテキストに引かれる取り消し線
- 左から右へ滑らかに描画される演出

### 5. スワイプアニメーション
- タスクを左右にスワイプして操作
- 右スワイプで完了、左スワイプで延期

### 6. 振動フィードバック
- タスク完了時の触覚フィードバック
- デバイスの振動機能を使用した物理的な反応

### 7. 統合タスクコンポーネント
- 全てのアニメーションを統合したタスクアイテムコンポーネント
- 一貫性のあるユーザー体験の提供

## 受け入れ条件

### チェックマークアニメーション
- [ ] タスク完了時にチェックマークが滑らかに描画される
- [ ] アニメーション時間が300ms以内
- [ ] 完了後にコールバックが実行される

### 波紋エフェクト
- [ ] タップ位置から波紋が広がる
- [ ] 600ms以内に完了する
- [ ] 色と透明度が自然に変化する

### スタンプアニメーション
- [ ] スタンプがバウンドしながら表示される
- [ ] スプリングアニメーションが適用される
- [ ] 回転を含む複雑な動きが実装される

### 取り消し線アニメーション
- [ ] テキストに取り消し線が左から右へ引かれる
- [ ] 400ms遅延後に200msで描画
- [ ] 取り消し可能（アンドゥ対応）

### スワイプアニメーション
- [ ] 右スワイプでタスク完了
- [ ] 左スワイプでタスク延期
- [ ] 100px以上のスワイプで反応
- [ ] スプリングバックアニメーション

### 振動フィードバック
- [ ] タスク完了時に振動が発生（実機）
- [ ] スワイプ時に軽い振動
- [ ] エラー時に異なるパターンの振動

### 統合タスクコンポーネント
- [ ] 全アニメーションが連携して動作
- [ ] パフォーマンスが60fps維持
- [ ] メモリリークがない

## 成功指標

- アニメーションが60fpsで動作すること
- ユーザーテストでポジティブな反応が得られること
- タスク完了率の向上（将来的に測定）

## スコープ外

以下はこのフェーズでは実装しません:

- カスタムアニメーションエディタ
- アニメーション速度の設定機能
- Lottieアニメーションの統合
- 複雑なパーティクルエフェクト
